# 🦞 OpenClaw AI-assistent — Setup Script

Interaktivt setup-script som skapar en personlig AI-assistent via OpenClaw.

## Användning

```bash
curl -fsSL https://raw.githubusercontent.com/filipthai94-hub/new-setup/main/setup.sh | bash
```

## Vad scriptet gör
- Frågar dig om namn, pronomen, syssla, botnamn, emoji, Discord guild-ID
- Skapar alla workspace-filer (SOUL.md, AGENTS.md, USER.md, etc.)
- Konfigurerar modeller, streaming och Discord-inställningar

## Krav
- [OpenClaw](https://docs.openclaw.ai) installerat
- Node.js 22+
- `.env`-fil med API-nycklar
- Discord-bot skapad och inbjuden till server
